# UI/UX Portfolio

Frontend-only UI/UX designer portfolio for Sudarshan Madewad.

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Notes

- Theme is persisted in localStorage.
- Project case studies are driven by data in `src/data/projects.js`.
